# CLAUDE.md — 뮤지엄로드

전국 박물관·미술관·건축물 정보를 한눈에 보는 반응형 웹사이트.
디자인은 https://nooyoustop.bhsystem.kr 스타일을 따른다 (히어로 + 섹션 뱃지 + 카드 그리드 + 협업 섹션).

## 원칙 (반드시 지킬 것)

1. **비용 0원**: 유료 API 금지. 지도는 딥링크만, 데이터는 JSON 파일, 배포는 Vercel 무료.
2. 모바일 우선 반응형.
3. 주소·전화·전시일정 같은 사실 정보는 웹 검색으로 검증 후 입력. 추측 금지.

## 기술 스택

- Next.js (App Router, 정적 빌드) + TypeScript + Tailwind CSS
- 폰트: Pretendard / 아이콘: lucide-react
- 데이터: `/data/places.json`, `/data/events.json` (DB 없음)

```json

```

## 지도 연결 (API 키 불필요)

```
네이버지도: https://map.naver.com/p/search/{장소명 인코딩}
구글지도:   https://www.google.com/maps/search/?api=1&query={장소명 인코딩}
```
주변 맛집·카페도 이름으로 네이버지도 검색 딥링크 자동 생성.

## 페이지

1. `/` — 랜딩: 히어로(검색바) → 통계 카운터 → 퀵메뉴(박물관/미술관/건축물/전시일정) → **17개 시도별 카드** → 진행 중 전시 → 추천 공간 → 협업 제안 → 푸터
2. `/region/[시도]` — 시도별 목록 (타입 필터)
3. `/place/[id]` — 상세: 이미지 → 관람시간·휴관일·요금·전화(tel: 링크) → **네이버지도/구글지도 버튼** → 오시는 길(교통편·주차) → 진행 중 전시 → 주변 맛집·카페·핫플
4. `/events` — 전시 일정 (진행중/예정, 시도 필터)
5. `/partnership` — 홍보·협업 창구: 문화공간 등록 / 콘텐츠 제휴 / 지역 협력 3카드 + mailto 링크

## 작업 순서

1. **데이터**: 17개 시도 × 3곳 이상 = 60곳+ places.json 작성 (웹 검색 검증), 진행 중 전시 20건+
2. **랜딩 페이지** 완성
3. **지역별 목록 + 상세 페이지** (지도 딥링크 실작동 확인)
